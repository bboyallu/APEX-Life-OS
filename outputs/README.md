# outputs/ — On-Demand Reports

Custom summaries, briefings, and answers generated by the AI on request.

- Each request produces a new, date-prefixed Markdown file
  (e.g. `2024-06-01-q2-briefing.md`).
- Outputs are append-only: previous outputs are never overwritten.

See [`../CLAUDE.md`](../CLAUDE.md) for the full system schema.
